### 발견된 문제

**문제 1 — 네이버 쿼리에 전체 주소를 붙임**
- `"위담한방병원 서울특별시 강남구 삼성로 402, 지하2층일부, 2~7층 (대치동)"` → 결과 없음
- `"위담한방병원 강남구"` → 결과 있음 (홈페이지 URL 반환)
- 원인: 층수·호수·괄호 등 세부 주소가 검색을 방해

**문제 2 — 시군구 추출 로직이 "구" 대신 "시" 먼저 잡음**
- 주소 "서울특별시 강남구..."에서 `서울특별시`를 먼저 잡아 쿼리가 `"병원명 서울특별시"`로 생성됨
- 결과적으로 모든 병원에서 최적 쿼리가 만들어지지 않음

**문제 3 — 카카오 place 페이지가 React 렌더링**
- `BeautifulSoup`으로 HTML 스크랩 시 동적 콘텐츠를 읽지 못해 0개

**문제 4 — blog.naver.com 전체 제외**
- 소규모 한의원 중 공식 홈페이지 없이 네이버 블로그만 운영하는 곳이 다수
- 블로그도 자칭 컨셉 추출 소스로 유효한데 필터에서 제외됨

### 실패 병원 분류 (샘플 20개 기준)
| 원인 | 수 | 설명 |
|---|---|---|
| 홈페이지 자체 없음 | 9개 | 네이버에 등록은 있지만 link 필드 비어있음 |
| blog.naver.com만 있음 | 3개 | 블로그 필터링으로 제외됨 |
| 쿼리 오류 (시군구 버그) | 전체 해당 | 강남구 대신 서울특별시로 검색 |
| 이름 파싱 실패 | 1개 | `재단법인위담 서울위담한방병원` 검색 안 됨 |

### 발견된 문제

**문제 5 — 카카오 Place 공개 JSON API 없음**
- `place.map.kakao.com/main/v/{id}`, `place.map.kakao.com/info/{id}` 등 모두 404
- 장소 페이지(`place.map.kakao.com/{id}`)는 React SPA → BeautifulSoup 불가
- 홈페이지 정보는 JavaScript로만 로딩됨
- **결론:** Playwright 없이는 카카오에서 홈페이지 URL 추출 불가

**문제 6 — 네이버 히트율이 낮음 (1.3%)**
- 1,747개 대상 중 23개만 발견 = 1.3%
- 샘플 조사 결과: 대다수(~75%)가 온라인 존재 자체 없음
- 나머지는 카카오 지도에만 등록되었고 홈페이지 없음

## 한계 — 올릴 수 없는 경우

아래 병원들은 어떤 방법을 써도 홈페이지 URL을 찾을 수 없음:
- 온라인 존재 자체가 없는 소규모 의원 (네이버·카카오 미등록)
- 네이버·카카오에 등록은 있지만 홈페이지/블로그 미연결
- 카카오에만 등록된 병원 (홈페이지 정보가 React SPA 안에만 있음 → Playwright 필요)

이런 병원들은 심평원 공공 데이터(전문의 자격, 의료기기 신고)만으로 기본 정보 제공.

## 추가 개선 가능 방법

| 방법 | 예상 효과 | 복잡도 |
|---|---|---|
| Playwright로 카카오 장소 페이지 렌더링 | +50~200개 추정 | 높음 (설치·속도) |
| 네이버 display=5로 늘리기 | 미미 (이미 3개 확인) | 낮음 |
| 네이버 쿼리 변형 (이름만, 이름+동명) | 소폭 | 낮음 |
