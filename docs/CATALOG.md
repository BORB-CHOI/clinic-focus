# CATALOG — 데이터·신호 실험 기록

> 분류 정확도·신뢰도에 영향을 준 **실측 발견**을 간단히 누적하는 로그.
> "어떤 신호를 더하니 무슨 효과가 있었나"를 날짜별로 기록한다. (설계·결정 근거는 각 트랙 문서/PR에.)

## 2026-05-30 · 다신호 교차검증 효과 — 신호 1종 추가만으로 신뢰도 상승

**가설**: 병원 자체사이트(자칭) 하나만으로는 "여러 진료를 다 한다"고 적혀 있어 주력이 한 곳으로
안 좁혀진다. 독립적인 다른 신호(블로그·후기)를 더해 **교차 일치**를 보면 진짜 주력과 신뢰도가 또렷해진다.

**방법**: 강남구 PoC. 자체사이트가 크롤된 848개 병원을 룰 기반(LLM 0회)으로 분류.
네이버 블로그(공식 검색 API, 합법·무료)를 848개에 크롤 → `NAVER#BLOG` 적재 → 동일 병원 재분류.
신뢰도 모델은 [confidence-missing-signals 결정](#) — 결손 신호는 점수에서 제외, present 신호끼리만 교차.

**결과** (등급 분포):

| 신호 구성 | 확실 | 추정 | 정보 부족 | 검색 노출후보(score≥70) |
|---|---:|---:|---:|---:|
| 자칭만 (1종) | 7 | 110 | 731 | 117 |
| **자칭 + 외부 블로그 (2종)** | **19** | **150** | 685 | **169 (+44%)** |

**해석**: 독립적인 외부 후기 블로그 1종을 더하니 52개 병원이 "정보 부족"에서 위로 올라왔다(+44%).
자칭과 외부 블로그가 **같은 주력을 가리키는** 병원에서 신뢰도가 오른 것 — 결손을 점수에서 빼고
교차 일치만 반영하는 설계가 의도대로 작동.

> **저자(authorship) 정정 메모**: 처음엔 병원 **자체 블로그(자기 글)** 를 블로그 시그널에 섞어
> **173(+48%)** 로 측정됐다. 그러나 자체 블로그는 자칭과 **같은 저자**라 독립 교차검증이 아니다 →
> 자칭(self_claim)으로 옮기고 블로그 시그널은 **외부 제3자 후기 블로그만** 남겼다. 정정 후 **169(+44%)**.
> 차이는 4건뿐 — 즉 **개선의 거의 전부가 진짜 독립 시그널(외부 블로그) 덕분**이었음이 확인됐다(자기 블로그 부풀림은 미미). 다신호 교차검증 효과가 더 단단히 입증된 셈.

**함의**: 카카오·네이버 플레이스 후기(25%, 별점 후기)까지 더하면 3종 교차로 "확실"이 더 늘 것으로 기대.
신호가 하나도 없는 병원은 score 0(후순위)로 두되 검색에서 사라지지 않게 한다(의료법 차별노출 회피).

**관련**: `be/scripts/run_classification.py`(웹사이트 비필수 분류), `be/scripts/crawl_external_all.py`,
[`task-queue.md`](plans/task-queue.md).

> ⚠️ **위 +44% 는 네이버 블로그 *키워드 검색* 기준 — 아래 오후 정제로 방법이 바뀌어 superseded.**

## 2026-05-30 (오후) · 자체사이트 정제 + 블로그 소스 교체 (네이버 키워드 → 카카오 place앵커)

**무엇이 바뀌었나** (3가지):
1. **자체사이트 노이즈 정제** — depth 크롤이 빨아들인 블로그 RSS 아카이브 덤프·에러/준비중·중복
   페이지를 페이지 단위로 제거(`_filter_noise_pages`). 강남 2133 재처리: 페이지 26,377→11,531
   (**56%↓**), blog 16,221→3,442(**79%↓**), 본문 40%↓. **재크롤 없이** 기존 raw 재처리 후 S3 적재.
2. **블로그 시그널 소스 교체** — 네이버 블로그 키워드 검색 → **카카오 place앵커 blog**.
   실측 비교(같은 병원 490개): 동일 URL **교차오염 네이버 16.78% vs 카카오 0.75%(22배)**,
   타 병원 동시언급(리스티클) 50.7% vs 11.2%. 네이버 키워드는 절반이 옆 병원/명단/구인글이라
   호스트 앵커가 못 됨 → 폐기. 카카오 place앵커가 권위 소스.
3. **후기 시그널** — 카카오 place앵커 후기(641) 추가. **네이버 플레이스 후기는 미수집(보류)** —
   회색지대 Playwright 18~25초/건이라 로컬 PC 크롤로 분리(스크립트 준비됨, 도착 시 증분 합류).

**왜 숫자가 내려갈 수 있나(정직성)**: 네이버 키워드(854)를 빼고 카카오 place앵커 blog(347)만
쓰면 블로그 시그널 커버리지가 준다. 대신 **남은 신호는 진짜 그 병원 글**이라 신뢰도가 정직해진다.
커버리지는 네이버 place앵커 후기(로컬 크롤)가 도착하면 회복된다.
